# Employee Management System

A full-stack Employee Management System built using:

- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat

## Features

- Admin Login
- User Login
- CRUD Operations
- Search Users
- Pagination
- BCrypt Password Hashing
- Profile Image Upload
- Email Notifications

## Tech Stack

Java, JSP, Servlets, JDBC, MySQL, HTML, CSS, Tomcat
